# VitalSync - Emergency Medical Handover Tool

A mobile-first web application for paramedics to log patient data 
and generate AI-powered crisis summaries in under 30 seconds.

## Live App
https://app-b8d220j05jwh.appmedo.com

## Problem
In busy ERs and disaster zones, critical patient information is lost 
during handover. VitalSync solves this with a 5-second readable 
crisis summary card.

## Features
- Patient logging with quick-tap inputs
- Real medical protocol mapping (ACS, Ottawa Rules, GCS, ABCDE)
- Automatic vital sign flagging (Tachycardia, Hypoxia, Hypertension)
- Color-coded severity banner (Critical/Urgent/Stable)
- GCS neurological assessment built in
